FAQ

常见问题

关于上上吉邻工的入驻、服务范围、安全保障与收费说明。

什么是上上吉邻工？

上上吉邻工（Upupgood）是一个家庭服务灵活用工平台，面向所有自由职业技能人员，不限家政，连接需要到家服务的客户与能够上门提供服务的个人。平台通过服务者名片、智能匹配、担保支付与 T+1 结算，为双方提供可信赖的交易环境。

如何入驻成为服务者？

在微信中搜索小程序 "上上吉邻工"，零门槛免费注册，完善服务者名片（简介、技能、证书、案例），约 5 分钟即可开张接单。

平台提供哪些类型的家庭服务？

围绕家庭场景，不限于家政。典型类目：家庭保洁（含深度保洁）、月嫂与育儿、家电维修、家居安装、搬家、上门教学（钢琴、绘画、学科辅导等）、上门美容美发、上门健身私教、宠物上门护理、家庭烹饪、家庭摄影等。任何能上门提供服务的自由职业技能人员都可入驻。

如何保障交易安全？

平台提供担保支付：客户付款先托管在平台，服务完成并客户确认后再进入结算；服务者资金在客户评价后 T+1 安全到账。下单、支付、服务、评价、结算全链路可追溯。

客户如何找到合适的服务者？

客户可按位置、技能、评分、价格筛选；平台基于社区围栏的智能匹配算法推荐附近合适的服务者，客户可通过服务者名片查看经验、证书与真实口碑后再下单。

上上吉邻工的核心能力有哪些？

九大核心能力：服务者名片、智能匹配、交易全链路、推广裂变、吉力推荐（VIP 增值曝光）、吉力值积分、优惠券体系、即时通讯、AI 智能体客服。

上上吉邻工收费吗？

服务者零门槛免费入驻，不收取入驻费用。平台按每笔成交收取基本的软件服务费，用于覆盖支付通道、系统运营、担保结算、风控与客服等成本——具体费率与计算方式以微信小程序 "上上吉邻工" 内公示说明为准。此外平台提供可选的 "吉力推荐" VIP 增值服务，帮助服务者获得更多曝光与竞争力分析。

如何让 AI 助手（Claude）帮我下单？

上上吉邻工提供官方 Agent Skill，在 Claude Code 等支持 Skill 的 AI 工具中一键安装。详见 AI Skill 章节，或直接下载 SKILL.md。

Agent Skill 支持哪些操作？

双端共 33+ 个 API 操作。
客户端（16 个）：服务发现、发布/取消需求、预约下单、查询/修改/取消/验收订单、生成小程序跳转链接、订单数量统计。
服务者端（17 个）：订单列表/详情/最新/数量、接单/拒绝、完成服务、调整订单金额、专拍订单提交/改价/撤回、AI 匹配分析、报名需求/取消报名。
双端共用同一登录态，AI 根据对话场景自动加载对应文档。

和其他家政平台有什么不同？

两点关键差异：（1）服务者范围更广——不限于传统家政人员，面向所有能提供到家服务的自由职业技能人员；（2）定位是服务者的事业伙伴而非单次撮合——提供名片、获客、接单、结算、口碑沉淀、VIP 增值与 AI 助手的全链路工具，帮助任何技能型自由职业者把手艺变成可持续的事业。
