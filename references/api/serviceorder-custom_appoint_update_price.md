# POST /api/serviceorder/custom_appoint_update_price

服务者在未支付前修改专拍订单价格（无需二次审核）。

**前缀**：`/api/`（需要登录）

## 请求体

```json
{
  "order_id": 98765,
  "new_price": 428.00
}
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| order_id | int64 | 是 | 专拍订单 ID |
| new_price | float | 是 | 新价格 (0, 800] |

## 响应

```json
{}
```

成功修改；双端聊天卡片会通过事件管道自动刷新。

## 约束

- 仅未支付状态的专拍订单可改价
- 新价格上限仍为 **800 元**
- 本接口不触发二次审核（商品名称没变）；如需改名需先撤回再重新提交
